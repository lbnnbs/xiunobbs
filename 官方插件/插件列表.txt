插件列表

反灌水(xn_antispawn)--一系列的反灌水措施，表单 HASH，POST 频次限制。
精华主题(xn_digest)--支持三级精华设置，用尾部图标标示精华主题。
版块三级分类(xn_forum_level_3)--增强到三级版块。
版块合并(xn_forum_merge)--合并版块下的主题，帖子。
友情链接(xn_friendlink)--友情链接管理，在首页右侧显示友情链接列表。
插入代码(xn_insert_code)--插入统计代码，页脚代码，广告代码。
每日IP限制(xn_ipaccess)--限制每日IP发帖量、注册量、文件上传个数、大小、连续发帖数、防止灌水。
逻辑删除(xn_logic_delete)--逻辑删除。
幸运踩楼(xn_lucky_post)--设置幸运的回复楼层，踩中者会收到踩中的提示，并且会显示当前进行的进度。
在线手册(xn_manual)--在线手册，支持章节编写，自定义顺序，手册数据单独表和目录存放，支持评论、删除。
手机注册(xn_mobile)--通过手机短信登陆、注册、找回密码。
管理增强(xn_mod_enhance)--编辑历史，删除历史，版主管理日志。
我的回帖(xn_mypost)--我的回帖。
二级导航(xn_nav_2)--如果要把BBS当做站点的一个频道，安装此插件，可以实现二级导航。二次开发必备。
导航栏更多版块(xn_nav_more)--导航比较多的时候，显示省略号，点击下拉菜单显示更多的版块。
QQ登陆(xn_qq_login)--QQ一键登陆，自动生成用户名，获取头像，完成登陆。
已读未读(xn_read_unread)--未读主题采用加粗表示。
屏幕阅读(xn_screen_reader)--为视力障碍者做的增强文字提示。
XiunoBBS搜索插件(xn_search)--XiunoBBS4采用独特的处理方式，支持中文高效搜索(请设置my.cnf中的ft_min_word_len=2)。
代码高亮(xn_syntax_hightlighter)-支持 HTML JS CSS PHP C JS C++ 等常用语法的高亮。
标签增强版(xn_tag)--强大通用的TAG系统，可以用来替代多维主题分类，可以对TAG进行分类，一个主题可以对应最多4个TAG。
官方主题：暗调背景(xn_theme_dark)--暗调背景，半透明效果，不建议新手基于该风格修改。
官方主题：蓝色背景(xn_theme_paopao)--最简单的主题风格制作方式。新手可以基于此主题风格直接修改。
官方主题：红色风格(xn_theme_red)--重新使用auto.cmd编译下就出来了，建议有开发能力的基于此风格二次修改。
UMEditor百度编辑器(xn_umeditor)--UMEditor是一款轻量级的百度编辑器，代码相对精简。
用户最新主题(xn_user_recent_thread)--在详情页右侧，加入用户最新发表的主题列表。
验证码(xn_vcode)--开启验证码，抵御机器灌水，可以为登陆、注册、找回密码、手机验证码指定是否开启验证码。
微信公众号(xn_wechat_public)--微信公众号自动登录,自定义菜单,需要认证过的微信服务号。
